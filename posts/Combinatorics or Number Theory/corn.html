Combinatorics or Number Theory?

Problem

Brian and Stewie are decorating an infinite street of house numbered $1, 2, \dots$ where every house is painted either Red or Blue. Stewie is a bit particular about on how the house must be painted. Let $R(n)$ and $B(n)$ be the number of divisors of $n$ which are painted Blue respectively. Stewie insists that the following rules must be followed.

  • If $n$ is a perfect square, then $R(n) = B(n) + 1$.
  • If $n$ is not a perfect square, then $R(n) = B(n)$.
Show that if house $x$ and house $y$ are of the same color, then house $x \cdot y$ must be Red.

Solution

Very interesting problem that one of my friends(he actually made this problem) gave me. I actually wanted to do a mostly combinatorics solution, but I ended up doing a pure number theory solution anyway. Either way, it's a very nice problem that used some topics I thought I'd never use.

Define a function: $$c(n) = \begin{cases} 1 && n \text{ colored red} \\ -1 && n \text{ colored blue} \end{cases}$$

Then, Stewie's conditions boil down to this: $$S(n) = \sum_{d \mid n} c(d) \begin{cases} 1 && n = k^2 \\ 0 && n \neq k^2 \end{cases}$$ Notice there exists a Dirichlet Convolution here, namely $S(n) = c * 1$. We can then perform a Möbius inversion, giving us: $$c(n) = \sum_{d \mid n} \left( S(d) \mu \left( \frac nd \right) \right)$$ Where $\mu(x)$ is the Möbius function. We now show that $c$ is multiplicative.

Consider the following: $$\sum_{i=0}^k S(p^i) \mu (p^{k-i})$$ The only $i$ that makes the inside function non-zero is $i=k$ and $i=k-1$, so we consider those to get $c(p^k) = S(p^k) - S(p^{k-1})$. We're basically done now, as $$c(p) = \sum_{d \mid p} \left( S(d) \mu \left( \frac pd \right) \right) = S(1) \mu(p) + S(p) \mu (1) = -1 \implies c(p)^k = (-1)^k$$ If $k$ is even, $S(p^k) = 1$ and $S(p^{k-1}) = 0$, which gives $c(p)^k = c(p^k) = 1$, and for $k$ odd we get $S(p^k) = 0$ and $S(p^{k-1}) = 1$ which gives $c(p)^k = c(p^k) = -1$, so $c$ is multiplicative.

Notice that when $n=p_1^{\alpha_1} p_2^{\alpha_2} \dots p_k^{\alpha_k}$, we have that $c(n) = c(p_1^{\alpha_1}) c(p_2^{\alpha_2}) \dots c(p_k^{\alpha_k})$. It's not hard to see that when $j$ is even, $c(p^j) = 1$ and when $j$ is odd, $c(p^j) = -1$. So we have that: $$c(n) = (-1)^{\text{number of odd }a_i's}$$ Alright, that's all we needed now, as $x$ and $y$ having the same color means $c(x) = c(y)$, and we have: $$c(x \cdot y) = c(x) \cdot c(y) = (-1)^{2(\cdots)} = 1 \implies c(x \cdot y) \text{ is red}$$ We're done. $\square$
